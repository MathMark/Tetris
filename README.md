![pieces](https://pp.vk.me/c628122/v628122649/38c99/6v25JfewLHo.jpg)

## Control

| Button | Function |
|-----------------------------------------|---------------------|
| &uarr;|Rotate block|
|&darr;|Shift block to down faster|
|&larr;|Move block to left|
|&rarr;|Move block to right|
|P|Set the game on pause|
|R|Restart the game|
|C|Reset the top score|
